####################
# feat : 새로운 기능 추가
# fix : 버그 수정
# docs : 문서 수정
# chore : 그 외 자잘한 작업
# test : 테스트 코드
# build : 시스템 또는 외부 종속성에 영향을 미치는 변경사항 (npm, gulp, yarn 레벨)
# ci : CI 관련 설정
# style : 코드 의미에 영향을 주지 않는 변경사항 (포맷, 세미콜론 누락, 공백 등)
# refactor : 성능 개선
# remove: 파일 삭제만 했을 때
# rename: 파일 및 폴더 이름만 바꿨을 때
####################
