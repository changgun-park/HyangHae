#### 로컬에서 실행하기

---

1. 전체 레포지토리를 클론한 후 FE 디렉토리로 들어와 주세요.

   ```
   git clone 'our repository url'
   cd FE
   ```

   

2. 패키지를 설치하고, 개발 서버를 구동합니다.

   ```
   npm i
   npm run dev
   ```
